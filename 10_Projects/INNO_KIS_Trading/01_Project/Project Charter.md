# Project Charter

## 1. 프로젝트 이름

INNO_KIS_Trading

## 2. 프로젝트 목적

이 프로젝트는 국내 주식 대상 투자 리서치, 정량 스크리닝, 관심 universe 관리, 규칙 기반 매매 시스템을 체계화하기 위한 개인 개발 프로젝트다.

본 프로젝트의 목적은 투자 수익률을 보장하는 것이 아니다.  
목적은 아래와 같다.

1. 국내 주식 전체 종목을 정량 기준으로 스크리닝한다.
2. 후보 종목에 대해 AI 리서치 프레임워크를 적용한다.
3. 투자 가능한 관심 universe를 구조적으로 관리한다.
4. 실제 진입과 청산은 검증 가능한 규칙 기반 시스템으로 분리한다.
5. 모든 데이터, 계산, 리서치 결과를 재현 가능한 형태로 저장한다.

## 3. 프로젝트 범위

### 포함 범위

- 국내 상장주식 대상 정량 스크리너 개발
- 국내 전체 종목에서 후보 30개 선별
- 후보 종목에 대한 AI Berkshire식 리서치
- 상위 5~10개 관심 universe 관리
- 기술적 진입 조건 설계
- ATR 기반 손절 규칙 설계
- 트레일링 스탑 규칙 설계
- 시장 국면별 비중 조절 규칙 설계
- 백테스트 및 실험 기록 관리
- 데이터 저장 정책 관리
- 리서치 리포트 Markdown 저장

### 제외 범위

- AI가 직접 매수 또는 매도 결정을 내리는 기능
- 투자 수익률 보장
- 검증되지 않은 데이터 기반 의사결정
- 출처 없는 재무 수치 사용
- 임의 생성 금융 데이터 사용
- 사람의 최종 판단 없이 자동으로 실거래를 실행하는 구조

## 4. 전체 시스템 구조

프로젝트는 아래 4개 Stage로 구성한다.

### Stage 1. 정량적 스크리너

국내 전체 종목을 대상으로 정량 조건을 적용하여 후보 종목 약 30개를 선별한다.

예상 기준:

- 유동성
- 시가총액
- 재무 안정성
- 매출 및 이익 성장성
- ROE, ROIC 등 수익성
- 부채비율
- 현금흐름 품질
- 밸류에이션
- 최근 가격 추세
- 거래대금

### Stage 2. AI 리서치

Stage 1에서 선별된 후보 30개를 대상으로 AI Berkshire식 정성 리서치를 수행한다.

분석 관점:

- 사업모델
- 경제적 해자
- 재무 품질
- 현금흐름
- 밸류에이션
- 경영진
- 산업 경쟁 구도
- 리스크
- 실패 시나리오

AI 리서치는 매수/매도 판단기가 아니다.  
AI 리서치는 투자 가능한 universe의 질을 높이기 위한 정성 필터로만 사용한다.

### Stage 3. 최종 선별

AI 리서치 결과와 정량 점수를 함께 검토하여 상위 5~10개 종목을 관심 universe에 등록한다.

최종 universe 등록은 자동 매수 신호가 아니다.  
등록 의미는 “추적 관찰 대상”이다.

### Stage 4. 매매 시스템

관심 universe에 포함된 종목만 대상으로 규칙 기반 매매 시스템을 적용한다.

구성 요소:

- 기술적 진입 조건
- ATR 기반 초기 손절
- 트레일링 스탑
- 시장 국면별 비중 조절
- 포지션 사이징
- 리스크 제한
- 백테스트
- 실거래 전 검증

## 5. 핵심 원칙

### 투자 판단 원칙

- 이 프로젝트는 투자 수익률을 보장하지 않는다.
- AI는 매수/매도 결정을 내리지 않는다.
- AI는 리서치 보조 도구로만 사용한다.
- 실제 진입과 청산은 검증 가능한 규칙 기반 시스템으로 분리한다.
- 최종 투자 판단은 사용자가 직접 수행한다.

### 데이터 원칙

- 금융 데이터는 임의 생성하지 않는다.
- raw data는 수정하지 않는다.
- 모든 핵심 수치는 출처를 기록한다.
- 출처가 불분명한 데이터는 사용하지 않는다.
- 불확실한 데이터는 “확인 불가”로 표기한다.
- 국내 주식 분석 시 DART, KRX, 기업 IR, 공시 자료를 우선한다.

### 개발 원칙

- 모든 계산 로직은 테스트 코드로 검증한다.
- 모든 스크리너 기준은 config 파일에서 관리한다.
- 데이터 수집, 데이터 가공, 리포트 생성, 매매 규칙은 역할을 분리한다.
- 결과물은 재현 가능해야 한다.
- 룩어헤드 바이어스를 방지한다.
- 생존자 편향을 방지한다.
- 과최적화를 방지한다.

### 리서치 원칙

- 리서치 결과는 Markdown 리포트로 저장한다.
- 투자 판단 문구는 “매수/매도 지시”가 아니라 “리서치 결과”로 표현한다.
- AI가 확신할 수 없는 내용은 추정하지 않는다.
- 모든 점수는 근거와 함께 작성한다.
- 정성 분석과 정량 분석을 분리해서 기록한다.
